# TaskFlow Pro — Менеджер задач

**TaskFlow Pro** — это веб-приложение для управления задачами с авторизацией, досками и системой приоритетов. Проект написан на чистом JavaScript с использованием `localStorage` для хранения данных.

 🚀 Функционал
- Регистрация и вход пользователей
- Создание досок с выбором цвета
- Добавление задач с приоритетом (Низкий, Средний, Высокий)
- Установка дедлайна и описания задачи
- Фильтрация задач по статусу (Все, Активные, Завершённые, Важные, Сегодня)
- Отметка задачи как выполненной
- Удаление и редактирование задач
- Статистика по доскам и задачам
- Адаптивный дизайн для всех устройств

 🛠 Технологии
- HTML5
- CSS3 (Flexbox, Grid, CSS-переменные, медиа-запросы)
- JavaScript (ES6+, `localStorage`, DOM-манипуляции)
